Tipos de Estructuras de Datos

Arreglos (Arrays)

Colección de elementos almacenados en posiciones de memoria contiguas. Acceso directo por índice.

Colas (Queues)

Estructura FIFO (First-In, First-Out). El primer elemento en entrar es el primero en salir.

Pilas (Stacks)

Estructura LIFO (Last-In, First-Out). El último elemento en entrar es el primero en salir.

Listas Enlazadas

Colección de nodos que contienen datos y referencias al siguiente nodo en la secuencia.

Tablas Hash

Estructura que implementa un array asociativo que mapea claves a valores mediante función hash.

Árboles

Estructura jerárquica con un nodo raíz y sub-árboles hijos. Ej: árboles binarios, AVL, B.

Grafos

Conjunto de nodos (vértices) conectados por aristas (edges). Pueden ser dirigidos o no dirigidos.

Ejemplos de Aplicación

Arreglos

Almacenar y acceder a datos de forma eficiente por índice. Ej: lista de estudiantes, pixels de una imagen.

Colas

Gestión de procesos en sistemas operativos, impresión de documentos, atención al cliente.

Pilas

Evaluación de expresiones, backtracking en algoritmos, historial de navegación (deshacer/rehacer).

Árboles

Organización de sistemas de archivos, DOM de HTML, algoritmos de compresión, bases de datos.

Grafos

Redes sociales, sistemas de navegación (GPS), routing de internet, recomendaciones.

Tablas Hash

Bases de datos, cachés, implementación de diccionarios, verificación de integridad de datos.

Comparativa de Estructuras

Estructura Ventajas Desventajas Complejidad Acceso
Arreglos Acceso rápido por índice Tamaño fijo (estáticos) O(1)
Listas Enlazadas Tamaño dinámico, inserción/eliminación eficiente Acceso secuencial O(n)
Pilas Útil para backtracking Acceso limitado (solo tope) O(1) para push/pop
Colas Órden FIFO justo Acceso limitado (solo frente) O(1) para enqueue/dequeue
Árboles Búsqueda jerárquica eficiente Complejidad de implementación O(log n) en balanceados
Grafos Modelan relaciones complejas Alto consumo de memoria O(V+E) para recorrido
Tablas Hash Búsqueda muy rápida en promedio Colisiones, orden no preservado O(1) en promedio
